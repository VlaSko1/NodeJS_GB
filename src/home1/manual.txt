Описание. 

Данная программа выводит в консоль простые числа, из указанного диапазона включительно. 
При этом числа окрашиватся в цвета по принципу светофора:

●	первое число выводится зелёным цветом;
●	второе — жёлтым;
●	третье — красным.
Диапазон, куда попадут числа, указывается при запуске программы, через пробел.



Правила работы с программой.

1. Для запуска программы в консоли вводим: "npm run home1 число1 число2", где
● число1 - начало диапазона,
● число2 - конец диапазона.

2. Возможные ошибки: 
● одно или оба числа не будут введены, 
● "число1" будет больше чем "число2",
● введенные параметры программа не сможет корректно преобразовать в число, 
● одно из введенных чисел будет меньше 0, программа вернет ошибку. 

3. Для корректной работы программы желательно указывать верхний диапазон не более 10000 - 15000. На свой страх и риск, и при 
наличии хорошего железа можно увеличить верхний предел диапазона поиска.
